# Estudos de POO em Java

Este repositório contém exercícios de estudo sobre **Programação Orientada a Objetos (POO)** utilizando Java.

## 📚 Conceitos praticados

- Classes
- Objetos
- Métodos
- Arrays de objetos
- Loop for
- For-each
- Entrada de dados com Scanner
- Cálculo de valor de estoque

## Exercício

Sistema simples de cadastro de produtos que:

- Permite cadastrar 5 produtos
- Armazena nome, preço e quantidade
- Calcula o valor total do estoque
- Mostra todos os produtos cadastrados

## Tecnologias

- Java

## 🎯 Objetivo

Fortalecer a base em Programação Orientada a Objetos.
